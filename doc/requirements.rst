Requirements
------------

If you follow the instructions in :doc:`installation` (using pip) most of
these requirements should be installed for you.

Python requirements:

* Python >= 2.6
* Argparse >= 1.2.0 (Or Python 2.7)
* CherryPy >= 3.2.2
* Mutagen >= 1.20
* SimpleJSON >= 2.6.0
* SQLAlchemy >= 0.7
